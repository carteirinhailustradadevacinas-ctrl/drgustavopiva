📱 Carteirinha de Vacinas Digital - Dr. Gustavo Piva
Uma solução tecnológica desenvolvida para modernizar o acompanhamento vacinal pediátrico. Este Web App (PWA) permite que pais e responsáveis gerenciem as vacinas de seus filhos de forma intuitiva, personalizada e com rigor clínico.

🎯 Finalidade
O projeto nasceu da necessidade de substituir as cadernetas de papel por uma ferramenta digital que esteja sempre à mão. Ele calcula automaticamente os prazos vacinais, considerando as particularidades de cada criança, como a idade gestacional.

✨ Principais Recursos
Cálculo Inteligente: Determina doses pendentes e atrasadas com base na data de nascimento e prematuridade.
Calendário Híbrido: Organização clara entre vacinas do SUS e da rede privada.
Interface White Label: O código foi estruturado para permitir a personalização total de cores, logotipos e marcas para diferentes profissionais da pediatria.
Tecnologia PWA: Pode ser instalado no celular sem ocupar espaço nas lojas de aplicativos (App Store/Google Play).
🛠️ Tecnologias Utilizadas
HTML5 / CSS3 (Variáveis dinâmicas para temas)
JavaScript Vanilla
PWA (Progressive Web App)
🎨 Personalização
O projeto suporta diversos temas visuais (Azul, Verde, Roxo, Rosa e Laranja), permitindo que cada pediatra tenha uma versão que reflita a identidade visual do seu consultório.
